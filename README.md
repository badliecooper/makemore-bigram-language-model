# Bigram Language Model from Scratch

Implementation of a character-level Bigram Language Model inspired by Andrej Karpathy's Makemore series. :)

## Concepts

- Probability Distributions
- Bigram Statistics
- Negative Log Likelihood
- Gradient Descent
- PyTorch

## Results

Generated realistic name-like sequences using learned character transition probabilities.

## Learning Outcomes

- Understanding probabilistic language models
- Training neural networks using gradient descent
- Character-level text generation
